Icon Resource

github v0.0.2

About

  • icons
    • customizable scalable vector graphics
    • may be scaled up or down to any resolution with no aliasing
  • icon stacking
    • provide a context in which icons appear on top of one another
    • icons inside this context can share the same scaling
  • when an invalid value is provided for an input
    • a descriptive error will be thrown
    • the last valid value will remain

Getting Started

  1. in your angular project run
    • npm install @joster-dev/icon --save
  2. in the module that utilizes these controls add
    • {{"import { IconModule } from '@joster-dev/icon'"}}
  3. in the module update your imports
    • imports: [..., IconModule]

Icon Inputs

  • type
    • expects {{types}}
    • must be specified
  • size
    • expects string
    • default '100%'
    • for scaling
  • fill
    • string | string[]
    • default 'FFFFFF'
  • fillRotate
    • boolean
    • default false
  • fillOpacity
    • number
    • default 1
  • stroke
    • string
    • default '000000'
  • strokeWidth
    • number
    • default 5
